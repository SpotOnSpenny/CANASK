## Requirements
Requirements are listed in the requirements.txt file, and can be installed from such.

** PLEASE NOTE **
The skDataPubCentre scraping function makes use of the package tabula-py in order to scrape the tabular data from a PDF file. This package requires Java to be installed. In Linux, this is as simple as running the following command in your terminal:
````
sudo apt install jre-default
````
This may differ for you depending on your operating system. Kindly look up the proper documentation for your operating system to install Java if you wish to run the skDataPubCentre scrape, otherwise, an error will be raised and the scrape for this source will not take place.
